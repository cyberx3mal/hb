# Birthday Quest — PWA

Коротко:
- React + Vite
- PWA манифест + простой service worker
- Вход по паролю (SHA256) — настрой в src/config.js
- Шаги и контент в src/data/steps.js
- Медиа — положи в public/media
- Финальное видео — укажи ссылку в src/config.js (YouTube unlisted)

Запуск локально:
1. npm install
2. npm run dev
3. Открой http://localhost:5173

Сборка:
- npm run build (потом можно залить на GitHub Pages / Vercel / Netlify)

Деплой:
- GitHub Pages: после билда содержимое dist заливаем в gh-pages branch либо используем gh-pages action.
- Vercel / Netlify: просто подключи репозиторий, они соберут проект автоматически.

Безопасность:
- Пароль в коде — клиентская защита. Не хранить секреты в публичном репозитории, если они должны быть действительно приватны.
- Большие видео загружай на YouTube/Vimeo как unlisted и вставляй ссылку.
